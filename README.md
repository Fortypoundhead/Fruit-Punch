Light and airy, FruitPunch is an XHTML and CSS compliant template. The inspiration was a display at a local farmers market. Just goes to show that inspiration can be anywhere!

Search engine optimization was thought of when building this template. As such, even though the menu is on the left, if you look at the code, the content comes first in the code. Three columns across the top, then a big content box below. Preview and Download links after the jump!

Since web crawlers read the site top-down, they'll hit the content first, and the more static menu entries next. Just helps you to get a bit of a boost, I think.

As always, this template is free for use by anyone. The only thing that I ask is that you tell me where you are using it, so I can see what good my coding is doing!
